# Sistema de Punto de Venta – Ferretería Cruz

## Descripción

Este proyecto consiste en una simulación de un sistema de punto de venta (POS) para la Ferretería Cruz.

La idea principal es mejorar la forma en que la ferretería maneja sus ventas, inventario y facturación, ya que actualmente todo se hace de manera manual, lo que puede causar errores y desorganización.

---

## Problema

La ferretería presenta varias dificultades debido a que no cuenta con un sistema digital, tales como:

* Registro manual de ventas
* Falta de control del inventario
* Errores en precios y cálculos
* Dificultad para generar reportes

---

## Objetivo

Desarrollar una simulación de un sistema de punto de venta que permita:

* Registrar ventas de forma organizada
* Controlar el inventario
* Generar facturas
* Facilitar el manejo de la información

---

## Funcionalidades

El sistema incluye:

* Gestión de productos (agregar, editar, eliminar)
* Control de inventario
* Registro de ventas
* Generación de facturas
* Visualización de reportes

---

## Cómo funciona (flujo básico)

1. El cliente realiza una compra
2. Se seleccionan los productos
3. El sistema calcula el total
4. Se registra el pago
5. Se genera la factura
6. Se actualiza el inventario

---

## Alcance

Este proyecto es una simulación visual, por lo que:

* No incluye pagos en línea
* No está conectado a una base de datos real
* Está pensado solo para uso académico
